# The Beam Grill & Lounge 

A fully interactive, single-page website for **The Beam Grill & Lounge** – a restaurant and lounge offering delicious food, live music, and entertainment.

![The Beam Grill & Lounge](https://i.postimg.cc/R0bT7jnF/menu1.jpg)

---

## Overview

This is a modern, responsive website designed to showcase the restaurant's brand, services, menu, and contact information. It features:

- **Home Page** – Hero section with stats and brand introduction
- **Services** – Displays key offerings: crafted food, mixology, live music, and games
- **Menu** – Clickable image gallery showing the full menu (lightbox for zoom)
- **Contact** – Contact details, social links, and a reservation inquiry form

The site uses a full‑screen background image, glass‑morphism effects, and a bold black‑and‑white color scheme for a clean, upscale feel.
